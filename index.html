wp-bullhorn

Bullhorn Integration Plugin for Wordpress

Features

  1. Custom post type for Bullhorn Job Order DTO
  2. CRON job to sync the Bullhorn DTOs periodically

Future Features

  1. Custom post type for Bullhorn Candidate DTO
  2. Custom taxonomy for Bullhorn Category DTO
  3. Ability for users to apply to jobs (public form for capturing Candidate data)
  4. CRON job to push the Candidate DTO entries to the server periodically

Installation

  1. Download the plugin directory and place in wp-content/plugins
  2. Activate the plugin
  3. Go to the plugin's settings page and fill in your Bullhorn API credentials
  4. Syncing data is currently configured to run as a wp_schedule_event.
    • Option 1: If you only have a limited amount of data saved in bullhorn, leave as an event.
    • Option 2: If you need have a lot of bullhorn data (causing timeout), OR if you cannot wait for an event to trigger* / need the update to happen at an inflexable time. Then set a system cronjob to execute `php /full/path/to/wp-bullhorn/cron.php`.
      * Note: wp events happen the next time a wordress request is made at the schedule time
